# Master's Salon Website

## Overview
This is a static website implementation for **Master's Salon**, featuring a modern luxury aesthetic.

## How to Run
Since this is a static HTML/CSS/JS site, you do not need to install any dependencies or run a build server.

1. Navigate to the project folder: `d:/Master Salon`
2. Open `index.html` in your web browser (Chrome, Edge, Firefox, etc.).

## Project Structure
- `index.html`: Homepage
- `services.html`: Services & Pricing
- `gallery.html`: Portfolio/Gallery
- `about.html`: About Us & Team
- `contact.html`: Contact & Booking Form
- `styles/`: CSS files (variables, reset, layout, pages)
- `scripts/`: JavaScript files

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Dark Luxury Theme**: High-contrast dark mode with gold accents.
- **Interactive Elements**: Hover effects, smooth scrolling, and mobile navigation.
